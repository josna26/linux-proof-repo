### whoami
Displays the username of the currently logged-in user.

### pwd
Shows the full path of the current working directory.

### mkdir <folder_name>
Creates a new directory with the given name.

### touch <file_name>
Creates a new empty file with the given name.

### rm <file_name>
Deletes the specified file from the directory.

### ls
Lists all the visible directories and files in the current location.

### ls -la
Lists all files along with the hidden files with detailed information like permissions and ownership.

### ls -a
Lists all the files in a directory, including the hidden files.

### cd <folder_name> 
Changes the current directory to the specified folder.

### mv <file_name> <path/of/new/location> (or) mv <file1_name> <file2_name>
Moves a file to a new location or renames it.

### top
Displays the real-time system processes along with the CPU and memory usage.
