### ðŸ”¹ touch <file_name>

### ðŸ”¹ rm <file_name>

### ðŸ”¹ ls
Lists all the visible directories and files in the current location.

### ðŸ”¹ ls -la
Lists all files along with the hidden files with detailed information like permissions and ownership.

### ðŸ”¹ ls -a

### ðŸ”¹ cd <folder_name> 
Changes the current directory to the specified folder.

### ðŸ”¹ mv <file_name> <destination_path> (or) mv <file1_name> <file2_name>

### ðŸ”¹ top
